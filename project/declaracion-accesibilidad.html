Declaración de accesibilidad

En la construcción del sitio web de Interacción Accesible (IAcoop) se han adoptado una serie de medidas con el objetivo de mejorar su accesibilidad. Esto conlleva una serie de ventajas como:

  • Facilitar el acceso de los usuarios independientemente de su condición física o de su entorno.
  • Incluir contenidos claros bien estructurados.
  • Mejorar la navegación y la experiencia del Usuario (UX).
  • Intrínsecamente mejora la usabilidad.

Cumplimiento de los estándares

Las páginas de este sitio web cumplen la marca AA según las Pautas de Accesibilidad para el Contenido Web 2.0 del W3C. Todos los requisitos han sido comprobados con un análisis manual de la accesibilidad a través de diferentes herramientas semi-automáticas.

Mensualmente se realizan revisiones automáticas de accesibilidad de este portal. La fecha de la última revisión manual sobre el nivel de accesibilidad es Setiembre de 2014.

Medidas adoptadas

  • Tecnologías de base: se utilizan los estándares HTML5 para marcar el contenido y CSS3 para el diseño visual.
  • Diseño y maquetación: separación de contenido y presentación, contraste entre los colores de texto y fondo, y uso de unidades relativas (permiten redimensionar el texto).
  • Alternativa a contenido no textual: se proporciona a las imágenes un texto alternativo mediante la propiedad "alt". En el caso de imágenes decorativas se deja la propiedad "alt" vacía.

Atajos de teclado

  • Ir al contenido principal de una página. (Atajo de teclado: 1)
  • Ir al mapa de sitio. (Atajo de teclado: 2)
  • Ir a la Declaración de accesibilidad. (Atajo de teclado: 3)

Uso de las teclas de acceso

  • Internet Explorer (Windows): ALT + Tecla + ENTER
  • Internet Explorer (Macintosh): Control + Tecla + ENTER
  • Mozilla (Windows): ALT + Tecla
  • Mozilla (Macintosh): Control + Tecla
  • Mozilla (Linux): ALT + Tecla
  • Opera 7.0 (Windows): MAYÚSCULAS + ESC + Tecla
  • Opera 7.0 (Macintosh): MAYÚSCULAS + ESC + Tecla
  • Opera 7.0 (Linux): MAYÚSCULAS + ESC + Tecla
  • Safari 1.2 (Macintosh): Control + ESC + Tecla
  • Konqueror 3.3 (Macintosh): Control + Tecla

Validadores utilizados

  • eXaminator
  • HERA
  • Validador W3C

Dudas o consultas

Cualquier sugerencia de mejora sobre la accesibilidad de este sitio web podrá hacerse mediante ir enviar un mensaje a través del formulario de la sección contacto.
